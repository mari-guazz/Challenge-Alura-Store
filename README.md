# 🛍️ Análise de Eficiência das Lojas - Alura Store

## 📌 Descrição

Este projeto tem como objetivo analisar os dados de vendas de quatro lojas da **Alura Store**, com foco em avaliar a eficiência de cada unidade. A partir dos dados fornecidos, realizamos uma série de análises para identificar **qual loja apresenta o pior desempenho** e, portanto, é a mais indicada para ser **vendida** pelo Sr. João, proprietário da rede.

## 📊 Análises Realizadas

O notebook contém análises exploratórias e comparativas baseadas em:

1. **Faturamento total por loja**  
2. **Distribuição de vendas por categoria de produto**  
3. **Avaliação média dos clientes por loja**  
4. **Produtos mais e menos vendidos**  
5. **Custo médio de frete por loja**

## 📁 Dados Utilizados

Os dados foram extraídos de arquivos `.csv` hospedados no repositório da Alura, com as seguintes colunas principais:

- `Produto`  
- `Categoria`  
- `Preço`  
- `Frete`  
- `Data da Compra`  
- `Avaliação`

## 🧠 Conclusão

Após a análise de todos os fatores, o projeto conclui qual loja é a **menos eficiente**, considerando o equilíbrio entre custo logístico, faturamento, satisfação do cliente e mix de produtos. Essa unidade é recomendada para ser vendida, liberando recursos para focar nas operações mais lucrativas.

- Pandas

- Google Colab / Jupyter Notebook
